A trait representing error values to be converted into an HTTP response.

Roughly speaking, this trait extends failure::Fail, a nearly standard of error abstraction in Rust, and provides the additional properties for handling the value as an HTTP response.

Note that this trait is defined as a sub trait of AsFail (not Fail itself), due to some restrictions of the current trait system.
